Summary on The Basics of the Web and HTML

Basics of the web

WWW

HTML pages that are linked by hyperlinks and communicated through HTTP protocol between the client and the server

HTML

HTML stands for Hyper Text Markup Language. HTML documents may contain text content which describes "what you see" and markup which describes "how it looks", also it may contain other content(media, files etc..) and links

HTML markup

HTML markup is used to spice up the plain text in HTML, HTML documents are made of HTML elements.Inside these elements contain tags. We also explored some of the tags that are used to make this page

Computers are Stupid

Computers are stupid because, they interpret instructions literally. This makes them very unforgiving since a small typo can cause huge problems in a program. no mistakes allowed!

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them
